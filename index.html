Introdução ao Modelo

O Modelo OSI foi proposto pela ISO como uma tentativa de padronizar os protocolos usados nas diversas camadas de comunicação. Ele organiza a comunicação em sete camadas, com o objetivo de dividir o processo de transmissão de dados em partes gerenciáveis e bem definidas.

Características das Camadas:

  • Abstração: Uma nova camada é criada quando há necessidade de mais abstração.
  • Função Específica: Cada camada realiza uma função definida.
  • Padronização: As funções visam garantir a compatibilidade dos protocolos.
  • Fronteiras Eficientes: Definidas para minimizar o tráfego desnecessário de informações.
  • Arquitetura Controlável: O número de camadas evita sobrecargas de funções.
